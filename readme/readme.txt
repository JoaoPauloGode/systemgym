Para que o sistema funcione a maquina deverá possuir:
*MySQL instalado
*Xampp instalado, e aplicativo de execução nesse diretório:
"c:\\xampp\\xampp_start"(caso não esteja nesse diretório o mesmo 
deverá ser alterado na classe Conexao do pacote dao)
===========================================================
*minha conexão possui senha, a mesma deve ser adaptada na string de
conexão com dataBase na classe Conexao
*caso o xampp seja iniciado naquele momento o sistema notificará que a
aplicação deve ser reiniciada para que funcione corretamente então deve-se
fechar a aplicação e executar novamente, isso só acontece quanto o xampp
é inicializado pela primeira vez
*passo a passo de como criar a tabela(criar um schema
chamado academia e na query dele executar o que tem no arquivo da pasta tabela)
